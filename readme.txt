------------ README -----------------

Pour démarrer le serveur et exécuter la démo :
	- start_server.sh
	- se rendre sur le localhost:8000/
	- sélectionner la page msprSimple.html

La page msprSimple contient la création du cube de base ainsi que la création des différents points correspondants aux coordonnées des autres cubes.

La partie concernant la recherche du flux vidéo dans les méthodes AR.js est manquante actuellement sur la master.
